# Apuntes de Bases de Datos

Proyecto web estático para GitHub Pages basado en HTML5, CSS3 y JavaScript Vanilla.

## Estructura

- `index.html`: documento principal con el contenido de la documentación.
- `style.css`: estilos modernos, sidebar fijo, modo oscuro y responsive.
- `script.js`: buscador instantáneo, cambio de tema, copiado de código y scroll top.

## Cómo usarlo

1. Abre `index.html` en el navegador.
2. Para publicarlo en GitHub Pages, sube esta carpeta a tu repositorio y activa Pages.
3. Si usas un subdirectorio, la URL será algo como `https://usuario.github.io/repositorio/`.

## Personalización

Puedes ampliar los ejemplos, añadir más temas o ajustar el diseño en `style.css`.

## Requisitos

- Navegador moderno.
- Sin dependencias externas.
- Compatible con GitHub Pages.
